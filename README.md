# chatApp
Chat Application using Nodejs and socket IO

# express install

npm i express

# Nodemon

npm i nodemon@1.18.9 --save-dev

# To run the project using dev command and configuration

npm run dev

# socket.io install for websocket

npm i socket.io@2.2.0
